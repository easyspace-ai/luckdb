| 📎 附件 | （可扩展） | 文件上传 |

## 🔍 技术细节

### Vite 配置
```typescript
export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, '../src'),  // 引用父目录
    },
  },
  server: {
    port: 3001,  // 独立端口
    open: true,   // 自动打开浏览器
  },
});
```

### 路径解析
```typescript
// 相对路径导入
import { Grid } from '../src/grid/core/Grid';

// 也可以使用别名
import something from '@/grid/...';
```

## 🐛 故障排查

### 端口冲突
如果3001被占用：
```typescript
// vite.config.ts
server: {
  port: 3002,  // 改成其他端口
}
```

### 热更新不工作
```bash
# 重启服务器
pkill -f "vite"
npm run dev
```

### 依赖问题
```bash
rm -rf node_modules
npm install
```
