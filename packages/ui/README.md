# @luckdb/ui

LuckDB UI Component Library - 可复用的 React 组件库

## 安装

```bash
pnpm add @luckdb/ui
```

## 使用

```tsx
import { Button } from '@luckdb/ui';

function App() {
  return (
    <Button variant="primary" size="md">
      点击我
    </Button>
  );
}
```

## 组件

- `Button` - 按钮组件
- 更多组件开发中...

## 开发

```bash
# 开发模式
pnpm dev

# 构建
pnpm build

# 类型检查
pnpm type-check
```
